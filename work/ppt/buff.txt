video API

方法
play() 开始播放
pause() 暂停播放
load() 重新加载媒体

属性
autoplay  是否自动播放
controls  手否显示播控控件
currentSrc	当前媒体的URL
currentTime	当前播放的时间，单位秒
defaultMuted	缺省是否静音
duration		媒体的播放总时长，单位秒
ended			是否播放结束
loop 			是否循环播放
muted 			是否静音
paused 			是否暂停
volume			音量值

事件
canplay 	当浏览器可以开始播放该视频时触发
durationchange	当媒体的总时长改变时触发
error 		当加载媒体发生错误时触发
pause 		当媒体暂停时触发
play 		当媒体播放时触发
volumechange	当前音量发生改变时触发
waiting		当视频因缓冲而停止时触发

web应用程序相关

1.WebSocket
在浏览器中通过http仅能实现单向的通信
要从服务器向客户端推送消息一般使用轮询
WebSocket实现了浏览器与服务器全双工通信


2.LocalStorage 本地存储

包括localStorage和sessionStorage
Cookie 4KB
LocalStorage 5MB 存储字符够用了

3.Web Worker
JavaScript 创建的Web程序，处理都是在单线程内执行
Web Worker 支持创建在后台运行的线程
